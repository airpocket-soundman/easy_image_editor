# フォトベンチ(Photo Bench)

スマホのブラウザで動く画像編集Webアプリ。外部ライブラリ・サーバー不要。画像は端末外に送信されません。PWA対応で、ホーム画面に追加すればオフラインでも使えます。

**Androidネイティブ版(APK)** も作成できます。ネイティブ版は端末内の写真を自動で一覧表示し、元のフォルダへ直接保存できます。ビルド手順は [SETUP_ANDROID.md](SETUP_ANDROID.md) を参照。

## 機能
- 写真を複数選択してサムネイル一覧から編集対象を選ぶ(静止画のみ。動画やGIFは除外)
- サムネイルのソート順を変更可(追加順・名前順・撮影/更新日時順)
- クロップ(四隅ドラッグ+移動)
- 90°単位の回転(左右)
- 自由角度回転(-180°〜180°、**内接**=余白なしに切り抜き / **外接**=全体を残す を選択可。どちらも縦横比は元画像のまま)
- 矩形の塗りつぶし(色選択)
- ペン書き込み(色・太さ変更、1筆ごとに確定)
- 矢印描画(ドラッグで作成後、先端・末端・本体をドラッグして回転/サイズ/位置を調整、色・太さ変更可)
- 元に戻す(直近6手)
- 別名保存(JPEG/PNG)

- 一度追加した写真はアプリ内(IndexedDB)に保存され、次回起動時に自動でサムネイル一覧に復元されます(×ボタンで一覧から削除可)
- Androidではギャラリーの「共有」→「フォトベンチ」で写真を直接開けます(共有ターゲット対応)

## 制限(ブラウザ仕様)
- 端末内の画像を自動スキャンすることはできないため、初回はフォトピッカーまたは「共有」で追加します
- スマホでは保存先は「ダウンロード」フォルダになります(元フォルダへの直接書き込みは不可)。PC版Chrome/Edgeでは保存ダイアログに元ファイルのフォルダが初期表示されます

## GitHub Pages での公開
1. このリポジトリの **Settings → Pages** を開く
2. Source: **Deploy from a branch**、Branch: **main / (root)** を選んで Save
3. 数分後 `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

## PWA(ホーム画面に追加)
- Android/Chrome: メニュー →「アプリをインストール」または「ホーム画面に追加」
- iOS/Safari: 共有ボタン →「ホーム画面に追加」
- 一度開けばオフラインでも起動できます(Service Workerがキャッシュ)
- `index.html` 等を更新したら `sw.js` の `CACHE_VERSION` を上げてください
